## Support & Troubleshooting

### Quick Fix Script Issues

**Problem:** "File not found" error  
**Solution:** Ensure you're in Project-root directory

**Problem:** "Backup already exists"  
**Solution:** Delete `*.audit_backup` files or merge changes manually

**Problem:** "Endpoint already exists"  
**Solution:** This is good! The fix was already applied.

### Audit Issues

**Problem:** Too many "unused routes"  
**Solution:** This is normal - see AUDIT_COMPLETE.md FAQ section

**Problem:** False positive API call detected  
**Solution:** Check enhanced_audit_report.json for the exact file/line

**Problem:** Audit takes too long  
**Solution:** Normal on first run; subsequent runs cache some data
